# knorrlabs

An independent lab for developer and infrastructure tooling, with a focus on Ignition, Kubernetes, and GitOps. Projects here are built to be used by people other than their author, so they live under a stable, impersonal home rather than a personal username.

### Focus areas

- **Ignition platform tooling** for deploying and operating gateways the GitOps way
- **Kubernetes operators and CLIs** that turn manual stack setup into declarative, repeatable workflows
- **Documentation** that treats DevOps and GitOps for industrial systems as a first-class concern

### Projects

| Project | What it is |
| --- | --- |
| [`stoker-operator`](https://github.com/knorrlabs/stoker-operator) | Kubernetes operator for managing Ignition stacks |
| [`ignition-stack`](https://github.com/knorrlabs/ignition-stack) | CLI for generating and running Ignition stacks |
| [`ignition-guides`](https://github.com/knorrlabs/ignition-guides) | Guides for DevOps and GitOps with Ignition |

> Project list is illustrative and will fill in as repositories are transferred in.
